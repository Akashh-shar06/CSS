CSS 2D Transform Hover Examples

Translate
Moves the element along X and Y axes: translate(50px, 30px)
Rotate
Rotates the element clockwise: rotate(45deg)
ScaleX
Scales the element horizontally: scaleX(1.5)
ScaleY
Scales the element vertically: scaleY(2)
Scale
Scales the element in both directions: scale(1.5)
SkewX
Skews the element along X axis: skewX(20deg)
SkewY
Skews the element along Y axis: skewY(20deg)
Skew XY
Skews the element along both axes: skew(20deg, 10deg)
Matrix
Matrix combines multiple transforms: matrix(a, b, c, d, tx, ty)
Example: matrix(1, 0.5, -0.5, 1, 50, 30)
